# Word2Pic Chrome 插件

## 项目概述

Word2Pic 是一款 Chrome 浏览器插件，用于将网页中的精彩文字内容转换为精美图片。用户可以选择不同的风格模板，将喜欢的金句、名言或重要内容一键生成为可分享的图片。

## 核心功能

### 基础版本功能

1. **文本输入**
   - 用户在网页中手动选择并复制文字内容
   - 在插件弹出窗口中粘贴所选文本
   - 支持手动编辑文本内容（调整、修改）

2. **图片生成**
   - 默认生成竖版图片，宽度固定，高度自适应文本长度
   - 提供 3-5 种基础风格模板（简约、文艺、商务等）
   - 支持预览生成效果

3. **图片导出**
   - 一键保存图片到本地
   - 支持 PNG、JPG 格式导出
   - 可复制图片到剪贴板

## 用户界面

1. **插件弹窗**
   - 简洁的文本输入框
   - 风格模板选择区
   - 预览窗口
   - 导出/保存按钮

2. **设计风格**
   - 界面简洁直观
   - 操作步骤清晰
   - 响应速度快

## 技术要求

1. **开发技术**
   - 前端：HTML, CSS, JavaScript
   - 可使用前端框架如 Vue.js 或 React
   - 图片处理：Canvas API 或 HTML2Canvas

2. **浏览器兼容性**
   - 支持 Chrome 最新版本
   - 兼容 Chromium 内核的其他浏览器（如 Edge）

## 安装与使用

1. 在 Chrome 网上应用店搜索并安装 "Word2Pic"
2. 点击浏览器工具栏中的插件图标打开
3. 复制网页中喜欢的文字，粘贴到插件输入框
4. 选择喜欢的风格模板
5. 点击生成按钮预览效果
6. 满意后点击保存或分享按钮
